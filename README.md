# Product Catalog

Одностраничное приложение на React + TypeScript, которое показывает список товаров, позволяет искать их по названию и открывать подробную информацию в модальном окне.

## Функциональность

- загрузка товаров из мокового API (`/api/products.json`)
- отображение карточек товаров
- поиск по названию в реальном времени
- открытие подробной информации о товаре в модальном окне
- закрытие модального окна:
  - по кнопке `X`
  - по клику вне окна
  - по клавише `Esc`
- адаптивный интерфейс для мобильных, планшетов и десктопов

## Стек

- React
- TypeScript
- Vite
- CSS Modules

## Запуск проекта

Установить зависимости:

```bash
npm install
```

Запустить:

```bash
npm run dev/build/preview
```
